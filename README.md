# api-register

### RESTful API for Sign-in/Sign-up.

### Featuring
  * DDD
  * TDD
  * TypeScript
  * MongoDB

## install dependencies:

### `npm install`

## config your `.env`

### based on file `.env.example`, create your `.env` with your information.

## run

### `npm start`

## run tests

### `npm test`
